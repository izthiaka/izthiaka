### Projet actuel

**Jolof Taxi** — Application mobile de réservation de taxi (Flutter + NestJS)

- Architecture microservices avec communication WebSocket temps réel
- Calcul automatique des tarifs (distance + temps + frais fixe)
- Système d'onboarding chauffeurs et suivi instantané des courses
- Objectif : expérience simple, rapide et transparente
